# Responsive QA Evidence Log

Use this file to store screenshot evidence for final submission.

## Viewports

- Mobile: `360x800`
- Tablet: `768x1024`
- Desktop: `1366x768`

## Capture Rule

- Take 3 screenshots per page (`mobile`, `tablet`, `desktop`).
- Save images under `Frontend/evidence/responsive/<page>/`.
- Naming format: `<page>-<viewport>.png`
  - Example: `student-dashboard-mobile.png`
