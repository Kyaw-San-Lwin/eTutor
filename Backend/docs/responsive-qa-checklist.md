# Responsive QA Evidence Checklist

Use this checklist to produce final coursework evidence for:

- mobile (360x800)
- tablet (768x1024)
- desktop (1366x768 or above)

## How to record

For each page:

1. Open in browser dev tools.
2. Capture screenshot for mobile/tablet/desktop.
3. Confirm:
   - sidebar/navigation usable
   - tables/cards readable
   - forms and buttons usable
   - no overflow/cut-off content

- Login
